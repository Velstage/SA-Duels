====================================================
SA DUELS — 페이지 구조
====================================================


----------------------------------------------------
1. 메인 페이지
----------------------------------------------------

· 캐릭터 목록 표시
· 캐릭터 선택 → 캐릭터 페이지로 이동
· 전체 페이지 진입 버튼


----------------------------------------------------
2. 캐릭터 페이지
----------------------------------------------------

· 캐릭터 프로필 (출신작, 전투 스타일, 플레이버 텍스트)
· 해당 캐릭터의 스킨별 이미지 갤러리


----------------------------------------------------
3. 전체 페이지
----------------------------------------------------

· 전체 캐릭터의 모든 이미지 표시
· 태그 검색 기능 — 각 이미지에 자유롭게 부착한 태그로 필터링
  (예: 스킨명, 패배 장면, 특정 복장 등)
